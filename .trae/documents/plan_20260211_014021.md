# 使用 Python 3.12.10 重新创建虚拟环境并安装 NeMo

## 步骤

### 1. 删除现有虚拟环境
```powershell
Remove-Item -Recurse -Force venv
```

### 2. 使用 Python 3.12.10 创建新虚拟环境
```powershell
py -3.12 -m venv venv
```

### 3. 激活虚拟环境
```powershell
.\venv\Scripts\activate
```

### 4. 重新安装 WhisperLiveKit
```powershell
pip install -e .
```

### 5. 安装 NeMo（用于说话人识别）
```powershell
pip install "git+https://github.com/NVIDIA/NeMo.git@main#egg=nemo_toolkit[asr]"
```

### 6. 运行服务器（带说话人识别）
```powershell
wlk --model-path "D:\python\FastWhisperTranscriber\model\models--Systran--faster-whisper-base" --language zh --backend-policy localagreement --backend faster-whisper --diarization
```

- Python 3.12.10 有 editdistance 的预编译 wheel
- NeMo 安装成功
- 服务器启动成功，支持说话人识别
